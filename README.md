# PinDial App 网站

这是 PinDial App 的官方网站仓库，用于托管 App Store 所需的隐私政策和技术支持页面。

## 页面列表

- **隐私政策**：`privacy.html` - App Store 审核必需
- **技术支持**：`support.html` - 提供常见问题和教程

## 本地预览

直接在浏览器中打开 HTML 文件即可预览。

## GitHub Pages 部署

1. 在 GitHub 上创建仓库（如 `pindial-app`）
2. 推送代码到 `main` 分支
3. 在仓库 Settings → Pages 中启用 GitHub Pages
4. 等待 1-2 分钟，页面将发布到：
   - `https://jerrynxk.github.io/pindial-app/privacy.html`
   - `https://jerrynxk.github.io/pindial-app/support.html`

## 更新页面

直接修改对应的 HTML 文件，提交并推送到 `main` 分支，GitHub Pages 将自动更新。
